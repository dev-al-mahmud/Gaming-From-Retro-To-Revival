Moby v2025.05.15

By Tracy Poff (2113) on May 18, 2025 · 2 comments

This week, we have updates to the rich text editor and various other small fixes and improvements.

  • The new rich text editor is now used for Company Trivia and Promo Image Group comments.
  • Spellcheck is now enabled on the rich text editor.
  • Official site links now open in a new window.
  • Posting to the forum will now scroll to your new post.
  • Forum reply notifications now scroll directly to the reply.
  • Approvers: the delisted games report now links directly to SteamDB.
  • Fix: large images in ad blurbs and official descriptions will now be scaled to fit the page.

Moby v2025.05.09

By Tracy Poff (2113) on May 09, 2025 · 2 comments

Another big(-ish) update this week: we will now document 'end dates' for game releases. These represent the date the game was no longer available, such as when it was pulled from a store. For more details, check our standards.

API:

  • Release end dates are now available for Bronze subscribers (request platforms.releases to see them).
  • You can now filter for delisted games. Add the parameter delisted=true and only games that (as far as we know) are not currently available for purchase will be returned. This is available to all API subscribers.

Others:

  • Releases can now have end dates set.
  • A filter has been added to the Most Wanted to restrict results to games that are no longer available for purchase. This won't be useful until we get end dates filled in.
  • Fix: The Nintendo.co.jp scraper is fixed, again.
  • Fix: The prices box no longer displays if there are no prices.
  • Fix: You can now see the eBay prices of games in your collection again.
  • Approvers: A report on possibly-delisted games missing end dates is available from the sidebar of the approval homepage.

Moby v2025.05.02

By Tracy Poff (2113) on May 02, 2025 · 6 comments

First update of the new month, and it's a good one. During April, MobyPlus users have been testing a new rich text editor, which makes it much easier to write formatted text and link to games. We've incorporated your feedback, and now the rich text editor is available to everyone! We'll continue to improve it, so please try it out (you can use it when commenting on this post, if you like) and let us know what you think.

Besides that, if you've been finding the red Atari banner to be a little eye-searing in dark mode, you'll be glad to know that we've added an option for MobyPlus users to hide the banners. Just visit your account settings (available from the user menu at the top right of every page) to set your preferences.

And finally, we've got another milestone to celebrate. Today, we added our 300,000th game: Despelote. That's 100,000 games added in just over four years. Amazing work, everyone!

MobyPlus:

  • Now MobyPlus users can hide banners. Visit your account settings to set this option.
  • The rich text editor that MobyPlus users have been testing is now available to all users. Thanks for all the feedback!

API:

  • Now the API will prefer more-specific release dates when reporting the first release of a game. For example, it will prefer 2005-11-21 to 2005.

Others:

  • Now the info block at the top of game pages will prefer more-specific release dates for the first release of a game, like the API.
  • The swipe-to-navigate behavior in image galleries has been tweaked. It should be harder to accidentally swipe now.
  • Fix: Now all non-US iTunes URLs in various formats) are accepted as Identifiers or when adding a new game.
